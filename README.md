# Tawa

Backend for the Kokopelli app and webplayer

## Running locally

First install all deps
```bash
npm i
```
Then running Tawa is as easy as running
```bash
npm run dev
```

## Configuration
Configuration is supplied through environmental variables or a .env file. Required variables:
* AUTH0_CLIENT_ID
* AUTH0_CLIENT_SECRET
* AUTH0_DOMAIN
* SESSION_SECRET
* AUTH0_CALLBACK_URL
